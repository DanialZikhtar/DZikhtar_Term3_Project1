Compile from command line: gcc -o LunarLander LunarLander.C -lm -lncurses

Controls:
W - Thrusters
A - Rotate Left
D - Rotate Right
Up - Thrusters
Left - Rotate Left
Right - Rotate Right
Esc - Exit Game
R - Restart game (after crash/landing)
